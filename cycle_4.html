For this assignment I was tasked with using an arduino board to program LED lights to do something when a button is pressed. Using tinker cad I created an arduino circuit that caused 3 LED lights to blink in a specific pattern with the first LED blinking upon power to the circuit being turned on.

When the button is pressed the arduino turns the middle LED on and doesn't turn it off until the other two LED's have blinked. I then took the arduino code from tinker cad to the arduino IDE to program a real arduino board and then build a physical version of my digital circuit.

After building the circuit in person I tested to see if it worked the same as the tinker cad version and it did.
